Pacman

Pacman was a lot of fun to develop. Everytime I work on it, there is something else I would like to add/modify. Give it try. Bring up a separate version in an incognito tab to have multiple sessions.

It is JavaScript/jQuery on the client side using socket.io to send and receive information from an Express server. The nginx server (serving the website brucelecaptain.com) is configured to do a proxy_pass to the Express server when a request is made to /pacman or /socket.io.
